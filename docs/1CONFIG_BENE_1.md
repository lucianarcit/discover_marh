# Configuração de Benefícios

## Descrição
Funcionalidade que permite à Empresa/RH habilitar as modalidades de benefícios (Refeição, Alimentação e Benefícios) que serão disponibilizadas para os colaboradores antes de realizar o primeiro pedido. Somente interlocutores com perfil de Decisão ou Gerenciamento podem efetuar essas configurações, e ao menos um benefício deve estar habilitado para que pedidos possam ser realizados.

**Nível de confidencialidade:** Uso interno

## Atenção

Antes de realizar o primeiro pedido, é necessário que a Empresa/RH habilite quais Auxílios serão disponibilizados para os colaboradores.

- Somente **Interlocutores de Decisão** e **Gerenciamento** conseguem fazer estas atualizações.
- A Empresa/RH poderá pular essa configuração, porém **se todos os Benefícios estiverem desabilitados, não será possível realizar um pedido**.

## Escolha dos Benefícios

A Empresa/RH escolhe os benefícios que vai disponibilizar ao colaborador e clica no botão para habilitar individualmente cada modalidade.

### Modalidades disponíveis:
- **Refeição**
- **Alimentação**
- **Benefícios**

> Atenção: essa tela apenas habilita benefícios para pedido. Ao habilitar os benefícios não significa que todas as opções selecionadas serão exibidas no aplicativo para o colaborador utilizar. É necessário fazer pedido com valor para que o colaborador possa utilizar os benefícios configurados nesta tela.

## Regras de Transferência

Não é permitida a realização de quaisquer transferências nas contas **Alimentação**, **Refeição** e **Benefícios**.

## Salvar a Configuração

Após configurar os benefícios, a Empresa/RH clicará em **"Salvar e continuar"**.

### Opções na tela:
- **PULAR** — ignora a configuração
- **SALVAR** — salva as configurações realizadas
