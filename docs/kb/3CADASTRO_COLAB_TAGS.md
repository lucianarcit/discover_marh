# Tags Produto — Colaboradores

## Descrição
Documento que explica o funcionamento das tags de produto associadas aos colaboradores no Sistema de Pedidos Alelo. A tag (ex: Tudo, Alimentação, Refeição, Benefícios) é atribuída automaticamente ao colaborador somente quando ele é vinculado a um pedido, e a tela de Colaboradores serve exclusivamente para pré-cadastro — não para configurar pedidos.

**Nível de confidencialidade:** Uso interno

## O que são as Tags de Produto

Após um colaborador ser relacionado a um pedido, ele receberá uma **tag de identificação de produto**.

> **Atenção:** A tag somente é associada se o colaborador for relacionado a um pedido.

## Tela de Colaboradores

A tela de Colaboradores serve **apenas para cadastro**. O cadastro nesta tela **NÃO configura pedido** — para isso, acesse a tela "Faça seu pedido".

### Ações disponíveis:
- **ADICIONAR NOVO COLABORADOR** — cadastro manual via tela
- **IMPORTAR COLABORADORES** — importação via planilha
- **EXCLUIR SELECIONADOS** — exclui os colaboradores marcados
- **FILTROS** — filtra a listagem

### Colunas da listagem de colaboradores:

| Coluna | Descrição |
|--------|-----------|
| Nome | Nome e CPF do colaborador |
| Data de Nascimento | Data de nascimento |
| Tipo | Tipo de entrega (Filial ou Residência) |
| Local de Entrega | Nome da empresa e CNPJ da filial vinculada |
| Produto | Tag indicando o produto associado (ex: **Tudo**) |
| Ações | Ícones para editar ✏️ ou excluir 🗑️ |

### Exemplo de colaborador cadastrado:
- **Bruna Cae** — CPF: 727.575.480-00 — Nascimento: 01/01/1999 — Tipo: Filial — Local de Entrega: Empresa Teste :: 42851375000105 — Produto: **Tudo**

## Importante

- A tela de colaboradores serve para **pré-cadastro** dos colaboradores
- A associação de produto (tag) ocorre apenas quando o colaborador é incluído em um pedido
- Produtos possíveis: **Tudo**, Alimentação, Refeição, Benefícios (conforme configuração da empresa)
