# Rastreio de Cartões

## Descrição
Funcionalidade acessível no menu Cartões > Rastreio de Cartões que permite à Empresa/RH acompanhar o status e o rastreamento de todos os pedidos de cartões, desde a produção até a entrega. Exibe listagem dos pedidos com número de rastreio, tipo de entrega e status, além de detalhe individual por AR com linha do tempo dos eventos, previsão de entrega, barra de progresso e possibilidade de busca por CPF do colaborador.

**Nível de confidencialidade:** Uso interno

## Como visualizar os status e rastreio dos cartões do colaborador

### Acesso
- Acesse o menu: **Cartões > Rastreio de Cartões**
- Selecione qual pedido deseja visualizar o detalhe do rastreio

### Atenção
- Caso a fabricação e entrega dos cartões **não tenha sido iniciada**, o botão de detalhe ficará **indisponível**

### Funcionalidades disponíveis

#### Listagem de pedidos
A tela de Rastreio de Cartões exibe uma listagem de todos os pedidos de cartões realizados e relacionados ao perfil do usuário.

Informações visíveis na listagem:
- **Produto** (ex: Alelo Pod)
- **Número do pedido**
- **Data do pedido**
- **Tipo de entrega** (Centralizada / Descentralizada)
- **Cartões emitidos**
- **Códigos de rastreio**
- **Status do pedido** (ex: Em processamento, Aguardando código de rastreio, Erro no processamento)

#### Botão "Ver Detalhes"
- Ao clicar em **Ver Detalhes**, é possível visualizar detalhes de **todas as AR's do pedido**
- É possível entrar no detalhe de cada entrega individualmente

#### Tela de Detalhes de Entrega
Permite filtrar o rastreio de cartões por:
- **Busca por CPF do colaborador**

Informações exibidas por entrega:
- **Número de rastreamento**
- **Local de entrega** (endereço completo com CEP)
- **Previsão de entrega**
- **Barra de progresso**: Produção → A caminho → Entrega
- **Responsável pelo recebimento** (nome completo)
- **Endereço de entrega**

#### Linha do tempo de eventos (Timeline)
Para cada entrega, é exibida uma linha do tempo com data, hora e descrição do evento. Exemplos de eventos:
- "Saiu para entrega"
- "Encaminhado para Centro de Distribuição / São Paulo-SP"
- "Encaminhado para Centro de Distribuição / Campinas-SP"
- "Em produção"

### Status possíveis dos pedidos
| Status | Descrição |
|--------|-----------|
| Em processamento | Pedido em processamento normal |
| Aguardando código de rastreio | Cartões ainda não despachados |
| Erro no processamento | Houve erro; entrar em contato com a central |

### Novidade no sistema
A tela de rastreio de cartões foi implementada para trazer mais controle e segurança nas entregas, permitindo acompanhar todo o processo com total autonomia.
