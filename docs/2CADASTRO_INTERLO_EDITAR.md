# Interlocutores — Cadastro e Edição de Usuários do Sistema

## Descrição
Funcionalidade que permite ao interlocutor de Decisão administrar os usuários do Portal Minha Empresa Alelo, incluindo cadastrar novos interlocutores em duas etapas (dados pessoais e dados empresariais), editar, bloquear e excluir usuários existentes. Cada usuário recebe um perfil de acesso (Gerenciamento, Operação ou Financeiro), sendo que o perfil Decisão é definido exclusivamente no contrato com a Alelo.

**Nível de confidencialidade:** Uso interno

## Acessando a Gestão de Interlocutores

**Caminho no sistema:** Menu **Administração** → **Usuários do sistema**

Na tela "Interlocutores | Usuários do sistema" é possível administrar todos os usuários que acessam o Portal Minha Empresa Alelo.

### Ações disponíveis na listagem:
- **ADICIONAR NOVO INTERLOCUTOR** — cadastra um novo usuário
- **FILTROS** — filtra a lista de usuários
- **IMPRIMIR** — imprime a lista

### Colunas da listagem:
- **Nome**
- **CPF**
- **Tipo de Usuário**
- **Ações** (editar ✏️ / excluir 🗑️ / bloquear 🔒)

---

## Cadastro de Novo Interlocutor

O cadastro é realizado em **2 etapas**:

### Etapa 1 — Dados do Interlocutor

Preencha os dados da pessoa que irá acessar o sistema e selecione o tipo de acesso desejado.

**Campos obrigatórios:**
- Nome completo
- CPF
- Data de nascimento
- Tipo de perfil (Gerenciamento / Operação / Financeiro)

### Etapa 2 — Dados Empresariais

Informações internas da empresa que facilitam a identificação da pessoa cadastrada.

**Campos obrigatórios:**
- Departamento
- E-mail
- Tel. Comercial

**Campos opcionais:**
- Tel. Secundário
- Ramal

### Botões:
- **CANCELAR** / **VOLTAR** — retorna sem salvar
- **AVANÇAR** — avança para a próxima etapa
- **SALVAR** — conclui o cadastro

---

## Observações

- Somente o interlocutor de **Decisão** pode adicionar e gerenciar outros interlocutores.
- O perfil **Decisão** não pode ser atribuído durante o cadastro pelo sistema — é definido no contrato com a Alelo.
